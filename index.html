Detailed Syllabus

Microsoft Excel Syllabus

  • Introduction to Excel for Data Analysis
  • Data Cleaning and Preprocessing
  • Advanced Formulas (VLOOKUP, HLOOKUP, INDEX/MATCH)
  • Pivot Tables and Charts
  • Data Validation and Conditional Formatting
  • Macros and Automation (VBA Basics)

SQL Syllabus

  • Introduction to Relational Databases
  • Basic SQL Queries (SELECT, FROM, WHERE)
  • Joins (INNER, LEFT, RIGHT, FULL)
  • Aggregations and Grouping (GROUP BY, HAVING)
  • Subqueries and Views
  • Database Design Principles

Power BI Syllabus

  • Data Transformation with Power Query
  • Modeling Data with DAX (Data Analysis Expressions)
  • Creating Interactive Reports and Dashboards
  • Publishing to Power BI Service
  • Row-Level Security and Data Governance
